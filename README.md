### What I'm building

**Unispect** — React Native property inspection app for a Calgary-based real estate firm. Offline-first SQLite with bidirectional sync to an Azure PostgreSQL backend, on-device media capture via VisionCamera, digital signatures, and a Node/Express/Puppeteer microservice that generates PDF reports. Shipped on the App Store and Google Play. I'm the sole mobile engineer on the project.

**Cheerfelt Circle** — Co-founder and CTO of an early-stage health-tech startup focused on care coordination between patients, families, and caregivers. Owning technical architecture and working toward launch with the founding team.
